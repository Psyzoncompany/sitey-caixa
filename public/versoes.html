Versão 6.0.43

Atual
  • Sincronização em tempo real sem recarregar página aplicada na Index: quando chega atualização remota, os cards e listas agora se redesenham automaticamente no mesmo layout.
  • O gatilho global de atualização remota foi migrado de reload da página para evento de refresh em memória, mantendo login protegido e evitando piscar de tela.
  • Mesmo comportamento estendido para Central, Clientes, Contas, Relatórios, Histórico e Configurações, garantindo atualização entre dispositivos sem F5 manual.

Versão 6.0.42

Anterior
  • Sincronização em tempo real entre dispositivos reforçada: quando chegar atualização remota da nuvem, as páginas do sistema agora se atualizam automaticamente sem precisar dar F5.
  • Foi adicionado gatilho de atualização apenas para mudanças vindas de outro dispositivo/sessão, evitando recarga desnecessária durante edição local.
  • A proteção da tela de login foi mantida para não forçar refresh enquanto o usuário ainda está autenticando.

Versão 6.0.41

Anterior
  • Refatoração do módulo pessoal com separação total: a página pessoal.html agora usa um script próprio (pessoal.js) baseado no script principal.
  • As gravações de dados no modo pessoal passaram para um JSON/chaves exclusivos com prefixo dedicado, evitando qualquer alteração cruzada no empresarial.
  • Navegação do pessoal mantida apenas com Pessoal, Contas e Metas, preservando a mesma interface visual do dashboard.
