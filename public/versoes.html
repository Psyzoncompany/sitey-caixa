Notas de Atualização

Histórico de melhorias e correções do site.

Versão 6.0.29

Atual
  • Corrigida a chamada da API Gemini no endpoint /api/chat para usar os campos corretos do payload multimodal (inlineData e mimeType), eliminando erro de validação quando houver imagem.
  • Estrutura de systemInstruction ajustada com role: system, melhorando compatibilidade com a API do Gemini hospedada na Vercel.
  • Histórico de conversa sanitizado para Gemini, convertendo conteúdos não textuais para texto válido e evitando falha quando mensagens anteriores vierem em formato de array/objeto.

Versão 6.0.28

Anterior
  • Correção do endpoint /api/chat do servidor Express para respeitar o modelo selecionado no botão da IA (Croq ou Gemini) em vez de sempre forçar Croq.
  • Adicionado retorno padronizado com content e modelUsed, mantendo compatibilidade com a tela ai.html e permitindo identificar qual IA respondeu.
  • Implementado fallback automático entre modelos apenas em limite/crédito (402/429), com aviso claro para o usuário quando a troca ocorrer.

Versão 6.0.27

Anterior
  • JSON enviado para a IA foi reforçado com o array pedidosDeCorte, trazendo cliente, peça, referência, total, cortado, pendente, status e prazo para cada pedido de corte.
  • A montagem dos dados agora lê os pedidos direto de production_orders e cruza com clients, garantindo que perguntas por cliente + referência retornem valores exatos de pendência.
  • Prompt da API foi atualizado para a persona Croq IA com regra explícita para responder cortes usando o JSON (incluindo soma quando houver múltiplos itens compatíveis).

Versão 6.0.26

Anterior
  • Prompt-base da API da IA foi atualizado para seguir o novo comportamento da assistente de negócio, com respostas objetivas em português do Brasil e uso explícito dos dados do sistema quando disponíveis.
  • Regras de resposta reforçadas para: não inventar dados, deixar claro quando faltar informação no contexto e priorizar análise de prazos, pendências e status quando a pergunta for sobre prioridades.
  • Tratamento de perguntas gerais mantido, permitindo responder assuntos fora do negócio sem perder o foco no contexto operacional quando ele existir.

Versão 6.0.24

Anterior
  • IA da página ai.html ficou mais completa com parte da inteligência reaproveitada do antigo chatbot.js, sem voltar com janela flutuante no painel.
  • Adicionado seletor de modo de resposta (Normal, Rápido e Profundo) para controlar o tipo de análise enviada à API.
  • Mensagens da IA agora possuem formatação melhor e ações rápidas contextuais para abrir páginas úteis (clientes, contas, processos e investimentos).
  • Implementado alerta proativo inicial com leitura dos dados locais (pedidos/saldo), deixando a IA mais inteligente para sugerir prioridade logo ao abrir.

Versão 6.0.23

Anterior

Versão 6.0.22
