Notas de Atualização

Histórico de melhorias e correções do site.

Versão 6.0.15

Atual
  • Sincronização do bloco de notas evoluída para tempo real com listener do Firestore, refletindo mudanças entre dispositivos sem precisar recarregar.
  • Atualizações remotas agora disparam evento interno para recarregar a UI de notas quando o usuário não está editando, evitando sobreposição de rascunho.
  • Link de Afazeres no dashboard corrigido para abrir a aba de processos, eliminando o erro 404 de página inexistente.
  • Referência do manifesto PWA ajustada para caminho relativo, reduzindo falhas de carregamento em alguns ambientes de deploy.

Versão 6.0.14

Anterior
  • Correção do bloco de notas na nuvem: alterações agora atualizam também o cache local do usuário antes do autosave.
  • Ao recarregar a página com internet instável, as notas recentes continuam disponíveis e voltam para a sincronização automática sem perder conteúdo.
  • Fluxo de persistência ajustado para reduzir risco de sumiço de dados entre refresh e envio ao Firestore.

Versão 6.0.13

Anterior
  • Bloco de Anotações agora abre em modo tela cheia em qualquer dispositivo, mantendo margens de segurança no mobile (safe-area).
  • Adicionadas abas no bloco de notas para criar múltiplas notas, alternar entre elas, renomear e excluir.
  • Rodapé do bloco ganhou ações rápidas (limpar, renomear, excluir) e indicador de último horário de atualização.

Versão 6.0.12

Anterior
  • Novo botão flutuante de Anotações, com janela estilo bloco de notas e salvamento automático para lembrar tarefas.
  • Empilhamento dos botões flutuantes ajustado: Afazeres acima de Anotações, e Anotações acima da Calculadora.
  • Sistema antigo de página de bloco de notas removido para eliminar o fluxo bugado.
  • Quadro de Processos ganhou filtro por prazo e agora sempre ordena os pedidos pelo vencimento mais próximo.

Versão 6.0.11

Anterior
  • Controle de Cortes agora separado em duas fileiras: "Cortando agora" e "Corte concluído".
  • Pedidos concluídos continuam acessíveis para abrir checklist e editar quando necessário.
  • Checklist de cortes recebeu filtros por status (Todos, Em andamento, Concluídos) e busca por texto.

Versão 6.0.10

Anterior
  • Correção no desktop: clique da calculadora agora abre o modal na mesma página, sem acionar navegação para outra tela.
  • Camadas (z-index) da calculadora elevadas para ficar acima dos elementos de navegação e evitar cliques indevidos no menu.
  • Comportamento no celular preservado, mantendo o fluxo já funcional.

Versão 6.0.9

Anterior
  • Removido o efeito de transição entre páginas para voltar ao comportamento original do site.
  • Correção da navegação inferior mobile: sem transformações globais no body para manter barra fixa estável.
  • Pré-carregamento de HTML em cache mantido, sem alterar a experiência visual na troca de tela.

Versão 6.0.8

Anterior
  • Sistema de salvar manual removido: toda alteração agora dispara sincronização automática com debounce na nuvem.
  • Transição suave adicionada ao trocar de páginas para evitar sensação de tela seca/carregamento brusco.
  • Pré-carregamento de páginas HTML habilitado em cache do navegador para reduzir espera na navegação interna.

Versão 6.0.7

Anterior
  • Cache local por usuário adicionado para manter os dados já carregados entre páginas sem reconectar do zero.
  • Ao entrar no site, a interface inicializa primeiro com cache local e sincroniza com o Firestore em segundo plano.
  • Salvamentos na nuvem agora também atualizam o cache local para preservar continuidade ao trocar de aba.
